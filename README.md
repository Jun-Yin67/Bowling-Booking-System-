# Bowling Booking System 

## Overview

The Bowling Booking System is a mobile application prototype designed to simplify the process of reserving bowling lanes. The system allows users to create accounts, manage bookings, make payments, redeem rewards, and access support services through a user-friendly interface.

This project was developed as part of a Human Computer Interaction (HCI) course project, focusing on user-centered design, usability, and interactive prototype development.

## Features

* User registration and login
* Password recovery
* Bowling lane reservation
* Booking management and tracking
* Payment processing
* Rewards and membership points redemption
* Notification system
* User profile management
* Help and support services
* About Us information page

## System Functions

### Account Management

* Sign up for a new account
* Sign in using Email, Gmail, Facebook, or Apple ID
* Reset forgotten passwords
* Update personal profile information

### Booking Management

* View available bowling lanes
* Reserve bowling lanes
* Select booking date, time, duration, and number of lanes
* View booking history and upcoming reservations

### Rewards System

* Earn membership points
* Redeem rewards and discount vouchers
* Claim rewards using redemption codes

### Notifications

* Receive booking confirmations
* View promotions and updates
* Manage notification preferences

### Customer Support

* Access FAQ and help resources
* Submit feedback
* Contact customer support through email

## Project Objectives

* Improve the bowling reservation experience through an intuitive interface.
* Reduce booking complexity and improve usability.
* Provide efficient booking and payment processes.
* Enhance user satisfaction through rewards and notification features.

## Technologies Used

* Axure RP (Interactive Prototype Design)
* Human-Centered Design Principles
* Usability Testing and Evaluation Methods
